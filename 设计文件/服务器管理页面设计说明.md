
## 1. 页面概览

- 页面名称：服务器管理
- 场景：管理 SSH/FTP 服务器列表，支持快速连接、编辑、删除。
- 终端：手机竖屏。
- 主要功能：
  - 列表查看已配置服务器
  - 搜索服务器
  - 快速选择连接协议（SSH / FTP）
  - 编辑 / 删除服务器
  - 新增服务器
  - 通过底部 Tab 切换到主页 / 设置

---

## 2. 布局结构（自上而下）

### 2.1 顶部导航栏

- 背景：主色蓝（与其他页面统一）。
- 左侧：汉堡菜单图标（打开侧边栏或其它模块）。
- 中间：标题文字 服务器管理，白色，居中。
- 右侧：放大镜图标按钮（全局搜索入口）。
- 状态栏时间/信号叠加在同色背景上。

### 2.2 面包屑 + 搜索

1）面包屑行

- 背景：浅灰条（与内容区区分）。
- 内容：
  - 左端：Home 图标。
  - 文本：我的服务器。
- 点击 Home 图标或文字，可返回上级首页（可选）。

2）搜索服务器

- 位于面包屑下方独立一行。
- 样式：浅灰背景圆角输入框。
- 左侧：放大镜图标。
- 占位文字：搜索服务器。
- 输入时过滤下方服务器列表。

### 2.3 服务器列表区域

- 背景：淡灰色（页面主背景）。
- 每一条服务器为卡片式列表项，竖向堆叠。

#### 列表项基本结构

- 卡片背景：白色，圆角矩形，有轻微阴影或边框。
- 左侧：服务器图标（机柜/主机样式）。
- 中间：两行文字垂直排列：
  - 第 1 行：服务器名称（如 DB Cluster 01、App Server），加粗。
  - 第 2 行：灰色辅文 SSH/FTP，表示支持协议。
- 右侧：状态 / 操作区域（根据不同状态变化）：
  - 默认：状态标签 + 状态点/文字  
    - 绿色状态点 + 在线  
    - 或灰色标签 离线 | 离线（图中为“离线/离线/高线”等可自定义，如“离线/异常/高延迟”等）
  - 选中服务器时：右侧变为蓝色操作区域（见「操作态」）。

#### 操作态（展开操作按钮）

- 当用户点击某条服务器卡片右侧区域时，该条卡片进入「操作态」：
  - 卡片右半部分背景变为深蓝色。
  - 显示两类操作：
    1. 左侧是快速连接协议列表：
       - 文本垂直排列：SSH、FTP。
       - 文本为白色，居中。
       - 点击 SSH：直接进入 SSH 连接（跳转 SSH 终端）。
       - 点击 FTP：进入 FTP 文件管理界面。
    2. 右侧为一个「更多」按钮：
       - 文本：更多（白色）或图标 ···。
       - 点击后弹出操作菜单。

#### 更多菜单（弹出层）

- 浮层从该卡片右侧向下弹出（白色小卡片，阴影）。
- 菜单项纵向排列：
  1. 编辑（左侧编辑图标）
  2. 删除（左侧垃圾桶图标）
- 点击空白处或再次点击卡片区域关闭菜单。

## 3. 底部导航栏

- 固定在底部。
- 左：HOME 图标 + 文本（或「首页」）。
- 中：圆形蓝色 + 按钮（浮动主操作）。
  - 点击：进入「新增服务器」页面（新建服务器配置表单）。
- 右：设置 图标 + 文本。
  - 点击：跳转到设置页面（如你前面那张设置图所示）。

## 4. 样式规范（建议）

### 4.1 配色

- 主色蓝：#2F7FD8（可按设计微调）。
- 背景灰：#F4F5F7。
- 卡片白：#FFFFFF。
- 主文字：#222222。
- 次文字：#888888。
- 状态点绿色：#2ECC71。
- 底部导航未选中文字：#777777。
- 操作态蓝色区域：主色的加深版（如 #2668B5）。

### 4.2 字体/字号

- 标题：16–18 px 半粗。
- 服务器名：15–16 px 半粗。
- 协议文字 SSH/FTP：13–14 px。
- 状态文字/按钮文字：13–14 px。
- 底部 Tab 文字：12–13 px。

### 4.3 间距/圆角

- 页面左右边距：16 px。
- 卡片圆角：10–12 px。
- 卡片间垂直间距：8–10 px。
- 底部中间 + 按钮直径：56 px 左右。

## 5. 交互与逻辑说明

### 5.1 搜索

- 输入关键字时，本地前端过滤服务器列表：
  - 匹配服务器名称（支持模糊匹配、大小写不敏感）。
- 输入为空：恢复全部列表。

### 5.2 列表项点击行为

- 点击卡片左侧/中间区域：
  - 可定义为「展开操作态」，或进入该服务器详情页（视产品需求）。
- 点击卡片右侧（状态区域）：
  - 进入「操作态」，显示 SSH / FTP / 更多。
- 再次点击其它卡片或空白处：
  - 收起当前操作态，恢复普通列表。

### 5.3 SSH / FTP 快速连接

- 点击 SSH：
  - 跳转到 SSH 终端页面，并携带当前服务器配置。
- 点击 FTP：
  - 跳转到 FTP 文件浏览页面（即你之前那张 FTP 文件管理 UI）。

### 5.4 更多菜单

- 编辑：
  - 打开编辑服务器配置页面，预填当前服务器信息。
- 删除：
  - 弹出确认对话框（“确认删除此服务器？”）。
  - 确认后从列表移除该服务器。

### 5.5 新增服务器（底部 + 按钮）

- 点击 +：
  - 跳转到「新增服务器」表单页（配置名称、地址、端口、SSH/FTP 账号等）。
- 保存后返回本页列表，并滚动到新建项位置（可选）。
